이종현 스타일리스트

부츠컷 팬츠 등 바지 스타일에 변화가 올 것 같다.
또한 반지부터 팔찌, 목걸이, 귀걸이까지
액세서리를 과하게 활용하는 스타일이
올해 큰 흐름이 될 듯하다.

비비드한 색감과 글리터 소재를 사용한
스타일이 올해 트렌드일 것 같다.
전체적인 젠더리스 실루엣에 원색의 아이템을 매치하고,
‘아디다스’하면 떠올릴 수 있는 이미지를
접목시켜 스타일링했다.

유정연 스타일리스트

트레이닝복을 단순히 운동용이 아닌 평상복으로 활용하는 요즘,
그저 편해서 입었다는 느낌이 들지 않게 스타일링했다.
밝은 트레이닝복에 코트나 가죽 재킷을 믹스 매치하고,
페미닌 한 드레스에 액세서리를 활용해
재미있는 룩을 완성했다.
